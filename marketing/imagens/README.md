# Banco de Imagens — Marketing Médico

Banco visual para posts, carrosséis e reels de saúde mental, psiquiatria e medicina do trabalho.

## Pastas

- `01-saude-mental/` — imagens conceituais sobre sofrimento psíquico, autocuidado, sono, ansiedade, depressão.
- `02-psiquiatria/` — transtornos específicos: bipolaridade, borderline, TDAH, dependência, jogos, psicose etc.
- `03-medicina-do-trabalho/` — burnout, saúde ocupacional, retorno ao trabalho, afastamento, ambiente corporativo.
- `04-fundos-para-texto/` — imagens limpas com espaço negativo para títulos e chamadas.
- `05-drone-salvador/` — imagens autorais/locais de Salvador, paisagens e identidade visual pessoal.
- `06-geradas-por-ia/` — imagens criadas por IA antes de curadoria final.
- `_referencias/` — referências visuais, moodboard e exemplos de estilo.
- `_selecionadas-para-postar/` — imagens aprovadas para uso imediato.

## Nome dos arquivos

Use nomes claros:

`tema-conceito-versao.ext`

Exemplos:

- `borderline-reflexo-vidro-01.png`
- `bipolar-luz-sombra-01.png`
- `burnout-mesa-trabalho-01.png`
- `ansiedade-janela-chuva-01.png`

## Critérios visuais

- Profissional, humano e acolhedor.
- Sem dramatização excessiva.
- Sem estética de “loucura”, perigo ou ameaça.
- Sem pacientes identificáveis.
- Sem jaleco/clichê médico quando não necessário.
- Preferir espaço livre para texto.
- Cores base: azul profundo, cinza, bege, branco, preto suave e tons quentes discretos.

## Uso ético

Evitar imagens que reforcem estigma em saúde mental. A imagem deve educar e acolher, não assustar.
