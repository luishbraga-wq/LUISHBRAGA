Aprofundamento do Bloco 6

Stack do curso, componente por componente.

Tudo escolhido pro empreendedor PME — Managed primeiro, VPS standalone fica como caminho avançado nos bônus técnicos. Cada componente tem papel específico, nada redundante.

1. Hostinger Managed OpenClaw

É onde seu agente vive. Equivalente a "alugar apartamento mobiliado" em vez de "comprar casa e construir do zero". Hostinger cuida de servidor, container Docker, rede, atualização do OpenClaw.

✓ Apenas OpenClaw Managed: R$ 30-50/mês (plano básico cobre 1 OpenClaw)

✓ OpenClaw + VPS standalone: R$ 50 + R$ 30-40 = R$ 80-90/mês (avançado · trilha R1-R4 nos bônus)

✓ Setup: 5 minutos no painel · cupom BRUNOOKAMOTO

✓ Limitação Managed: 1 OpenClaw por plano (multi-agente vive todo no mesmo workspace)

✓ Suporte: chat com agente Hostinger no painel · resposta <5min

Por que NÃO VPS standalone no curso? VPS exige conhecimento de Docker, Linux, Tailscale, debug de container. Aluno PME que nunca usou terminal trava no setup. Managed elimina 90% das pegadinhas técnicas. Se você é dev, bônus técnico R1-R4 cobre VPS — mas não é pré-requisito.

2. ChatGPT Plus (OAuth via terminal)

É o cérebro do seu agente — modelo principal (GPT-5.5 ou 5.4) usado pra conversas, planejamento, execução de tarefas complexas. Por que assinatura ChatGPT em vez de só pagar a API? Porque OAuth via Codex usa a quota da sua assinatura — fica significativamente mais barato em workflow agentic intenso.

✓ Pra começar: R$ 100/mês (ChatGPT Plus padrão · cota suficiente pra uso PME normal)

✓ Ideal pra uso intenso: R$ 550/mês (ChatGPT Pro · cota maior · necessário se você roda agente 24/7 com crons pesados, multi-agente, processamento de documentos longos)

✓ Setup: 1 comando no terminal Managed (cobertura na A1 Fase B)

✓ NÃO use mini ou 4o-mini em workflow agentic — modelos pequenos quebram em raciocínio multi-step

Estratégia do iniciante: começa com ChatGPT Plus (R$ 100/mês) · roda 30-60 dias · vê se a cota tá sendo suficiente. Se você travar em rate limit toda semana, migra pro Pro. Se não, fica no Plus pra sempre — maioria dos PME não precisa do Pro.

3. OpenAI API key (R$ 30-40/mês)

Complemento técnico essencial. A assinatura ChatGPT (OAuth) cobre conversas e raciocínio, mas Whisper (transcrição de áudio), embeddings (memória semântica) e image generation (DALL-E) só rodam via API key direta.

✓ Custo: R$ 30-40/mês (uso PME normal)

✓ Carga inicial: US$ 5-10 (cobre as primeiras semanas) · você recarrega quando o saldo cair

✓ Setup: conta em platform.openai.com (NÃO chatgpt.com) · cartão · crédito · API key

✓ Bônus oculto: serve como fallback nativo do OAuth — se OAuth falhar, agente cai automaticamente na API key

Pegadinha clássica: aluno PME confunde platform.openai.com (onde você gera API key) com chatgpt.com (onde você assina ChatGPT Plus). São SITES DIFERENTES. Se você cadastrou cartão no chatgpt.com pensando que era pra API, abriu assinatura — não API. Cancela e vai pro platform.openai.com.

Fase B — Ativar OAuth ChatGPT (segundo passo crítico)

Aqui você abre o terminal Managed pela primeira (e provavelmente única) vez. No curso V2, esse é o ÚNICO comando que você cola manualmente — daqui em diante o agente faz tudo via Telegram.

Terminal Managed cola e dá Enter
$openclaw models auth login --provider openai-codex --set-default
▶ Resultado: abre fluxo OAuth no navegador · você loga com sua conta ChatGPT Plus · agente passa a usar GPT-5.5/5.4 da sua assinatura · API key fica como rede de segurança automática.

Por que esse comando é manual (e não delegado pro agente)? Porque ele muda permissão de execução do agente — princípio defensivo do OpenClaw: comandos de elevação de privilégio NUNCA passam pelo agente. Você decide. Aprofunda na A10.

Princípios do curso

7 frases pra colar na geladeira.

Princípios transversais que aparecem em todas as aulas. Se você esquecer tudo do curso e lembrar só desses 7, ainda tira valor do agente.

1. Seu agente lê documentação.

Quando ele não souber, pergunta pra ele — ele vai na doc oficial OpenClaw e descobre. Você não precisa decorar nada técnico.

2. Contexto é tudo.

Quanto mais você dá (conversas, arquivos, decisões, dados), mais real ele fica. Não é configuração única — é alimentação contínua.

3. Trate como funcionário, não conta.

E-mail próprio quando faz sentido. Permissão por papel. Não é extensão da sua conta pessoal. Aprofunda na A10 (segurança operacional).

4. Memória = mapas mentais.

Você ensina o agente a se achar. Workspace organizado com MAPAs distribuídos. Aprofunda na A6 (workspace).

5. Vitória visível em toda aula.

Algo na tela que prova que funcionou. Se uma aula passar sem você ver resultado, alguma coisa quebrou — usa o grupo do Telegram.

6. Salve antes de limpar.

Higiene de contexto antes de /new ou /compact. Senão você perde decisões importantes. Aprofunda na A7 (memória).

7. Telegram é o canal default.

WhatsApp dá pra conectar mas tem ressalvas (A4 e A10). Slack/Discord/iMessage entram opcionais (A11 — 26 canais nativos).
