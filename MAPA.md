# MAPA — Workspace do Diabinho

> Este arquivo é o mapa raiz do workspace do Diabinho, agente do Henrique.

## Estrutura principal

- `content/` — coisas que eu crio para você: drafts, posts, emails, roteiros, materiais.
- `memory/` — decisões, projetos e contexto que preciso lembrar entre sessões.
- `skills/` — habilidades modulares instaladas no agente.
- `archive/` — itens antigos, substituídos ou obsoletos. Arquivar antes de deletar.

## Regras de uso

- Conteúdo novo vai para `content/drafts/` por padrão.
- Decisões importantes vão para `memory/decisoes/{YYYY-MM}.md`.
- Projetos ativos vão para `memory/projects/{nome}.md`.
- Cada pasta principal tem seu próprio `MAPA.md` local.

## Atualizar este mapa quando

- Criar uma nova pasta de primeiro nível.
- Mudar convenção de nomes.
- Adicionar um fluxo recorrente importante.
