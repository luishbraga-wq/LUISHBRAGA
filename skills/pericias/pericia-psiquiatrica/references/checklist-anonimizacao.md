# Checklist de anonimização — perícia psiquiátrica

Antes de processar caso concreto, confirmar que foram removidos ou substituídos:

- Nome completo do periciando
- CPF, RG, CNH, CNS, matrícula
- Número do processo
- Endereço, telefone, e-mail
- Nome de familiares identificáveis
- Nome de empresa, hospital, clínica ou escola quando identificar o caso
- Fotos, assinaturas, carimbos, QR codes
- Links públicos para documentos do caso
- Trechos de prontuário com dados identificáveis

Substituições recomendadas:

- Periciando: `[PERICIANDO]`
- Processo: `[PROCESSO]`
- Empresa: `[EMPRESA]`
- Local: `[LOCAL]`
- Datas: manter só se forem clinicamente relevantes; se não, usar `[DATA]`
- Profissionais: `[MÉDICO_ASSISTENTE]`, `[PSICÓLOGO]`, `[ADVOGADO]`

Se houver dado identificável, pare e peça versão anonimizada.
